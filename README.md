# Sistema de Gestão de Projetos e Equipes

## Descrição

Este projeto consiste no desenvolvimento de um sistema para gestão de projetos e equipes, utilizando a linguagem Java e os conceitos de Programação Orientada a Objetos.

O sistema tem como objetivo organizar e controlar informações relacionadas a usuários, projetos e equipes, facilitando o gerenciamento das atividades.

---

## Funcionalidades

- Cadastro de usuários
- Cadastro de projetos
- Cadastro de equipes
- Associação de usuários às equipes
- Exibição das informações cadastradas

---

## Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)

---

## Estrutura do Projeto

- Usuario.java
- Projeto.java
- Equipe.java
- Main.java

---

## Como Executar

1. Abrir o projeto em uma IDE Java (IntelliJ ou Eclipse)
2. Compilar os arquivos
3. Executar a classe `Main`

---

## Versionamento

O projeto foi versionado utilizando Git e hospedado no GitHub, permitindo o controle de alterações durante o desenvolvimento.
